# Prompt Templates(即コピペ用)

Claude Code に投げる定型プロンプトの雛形。`<...>` を埋めてそのまま貼り付けます。スラッシュコマンドにするほどでもない、状況依存の依頼に使います。

## 一覧

| ファイル | 用途 |
|---|---|
| `new-feature.md` | 新機能の実装依頼(計画 → 実装 → 検証) |
| `bug-investigation.md` | バグ調査(再現 → 原因特定 → 修正) |
| `perf-investigation.md` | パフォーマンス調査(計測 → ボトルネック → 改善) |
| `security-review.md` | セキュリティレビュー依頼 |
| `pr-description-jp.md` | PR 説明文の生成(日本語) |

## 使い方

1. 該当ファイルを開き、`<...>` を具体値に置換。
2. 本文を Claude Code に貼り付ける。
3. 必要に応じて関連スキル/コマンド/エージェントが自動起動する。
