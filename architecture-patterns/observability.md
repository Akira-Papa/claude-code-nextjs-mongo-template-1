# 観測性(ロギング / 監視)の方針

## ロギング

- `console.log` 直書き禁止。`lib/logger.ts` の `logger` 経由で構造化ログ。
- ログは「メッセージ + メタデータ」。秘密情報(トークン・PII)を含めない。
- レベル: `error` / `warn` / `info` / `debug` を用途で使い分ける。
- ユーザー向けメッセージと内部ログを分離(内部構造を画面に出さない)。
